ポーターの5フォース分析 対話型チャットボット 設計書
1. 目的

ユーザーに対して、ポーターの5フォース分析（業界内の競争、買い手の交渉力、売り手の交渉力、新規参入の脅威、代替品の脅威）を対話形式で実施し、事業環境を整理・可視化する支援を行う。

2. 主な要件
区分	内容
対象ユーザー	事業企画担当者、中小企業の経営者、コンサルタントなど
実施形式	5つのフォースごとに段階的な質問を提示し、会話形式で深掘り
利用環境	Webアプリ（例：Streamlit, Flask + Reactなど）、DifyなどのLLMエージェント環境でも可
言語	日本語（カジュアル・敬語なし）
出力	テキストベースの5フォース分析サマリー、グラフ化（オプション）
LLM連携	ユーザー回答から要約生成・補足質問提案・最終分析コメント作成
3. シナリオ・対話フロー概要（簡略版）
START
└──→「ポーターの5フォース分析を始める？」
        ↓
  ┌──→【1. 業界内の競争】
  │     ├─ 業界内に同じような競合は何社くらいある？
  │     ├─ 競合との差別化ポイントはある？
  │     └─ → 要約・次へ
  │
  ├──→【2. 新規参入の脅威】
  │     ├─ 新しくこの業界に参入しやすい？
  │     ├─ 参入障壁（規制、コスト、ブランドなど）は？
  │     └─ → 要約・次へ
  │
  ├──→【3. 代替品の脅威】
  │     ├─ 顧客は他の手段で代替できそう？
  │     ├─ コストや機能面で代替の強みはある？
  │     └─ → 要約・次へ
  │
  ├──→【4. 買い手の交渉力】
  │     ├─ 顧客は少数？大手？交渉力はある？
  │     ├─ 顧客が価格交渉してくる頻度は？
  │     └─ → 要約・次へ
  │
  └──→【5. 売り手の交渉力】
        ├─ 材料・部品・情報の供給元の数は？
        ├─ 取引先から値上げされることは？
        └─ → 要約・全体まとめへ

↓
【AIによる5フォースまとめ + コメント + 戦略アドバイス】

4. 機能仕様
機能	内容
フォース別QA	各フォースに3〜5問程度の質問（分岐なし、基本一本道）
ユーザー入力補助	選択肢、例文、参考リンクの表示
AIコメント	各セクション完了時に、回答を要約し補足分析コメント
全体まとめ	5フォースすべて完了後、全体のサマリーと戦略的視点を提示
保存・エクスポート	分析結果をテキスト/Markdown/PDF形式でダウンロード可能（任意）
5. 技術構成案（例）

フロントエンド：Streamlit, Next.js（選択可能）

バックエンド：FastAPI / Flask

AIエンジン：OpenAI GPT-4 or Claude 3, Dify Agent（選択）

状態管理：セッション/クッキー or DB（PostgreSQLなど）

出力管理：Markdown/PDF/CSVエクスポート機能（オプション）
